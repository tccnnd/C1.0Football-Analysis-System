#### 2. 简化工具栏 ✅

**移除的按钮**:
- ❌ "模型状态" → 移到菜单栏
- ❌ "用户中心" → 移到菜单栏
- ❌ C1统计按钮组（7个按钮） → 移到状态栏显示

**保留的按钮** (8个核心按钮):
- ✅ 刷新赛事
- ✅ 分析选中
- ✅ 分析全部
- ✅ 自动回收赛果
- ✅ 只看正式建议
- ✅ 只看待处理
- ✅ 只看接近阻断
- ✅ 只看阻断
- ✅ 恢复全部

**优化**:
- 添加分隔符区分功能组
- 保留C1筛选下拉框
- 保留自动轮询配置

#### 3. 优化状态栏 ✅

**新布局**:
```
[主状态信息] | [C1统计] | [C1模式] | [筛选状态]
```

**紧凑显示格式**:
```
C1: 正式5 待处理3 阻断1 | 显示8/10
```

**移除**:
- 原来的独立C1统计按钮组

#### 4. 添加窗口引用 ✅

**新增属性**:
```python
self.model_center_window: tk.Toplevel | None = None
self.analysis_center_window: tk.Toplevel | None = None
self.c1_center_window: tk.Toplevel | None = None
```

#### 5. 添加占位方法 ✅

**新增方法**:
- `open_model_center()` - 模型中心占位
- `open_analysis_center()` - 分析中心占位
- `open_c1_center()` - C1管理中心占位
- `_show_about()` - 关于对话框
