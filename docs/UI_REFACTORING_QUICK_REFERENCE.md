## 常见问题

### Q1: 找不到"模型状态"按钮了？
**A**: 现在在菜单栏 → 工具 → 模型状态总览

### Q2: 找不到"用户中心"按钮了？
**A**: 现在在菜单栏 → 功能中心 → 用户中心（传统）

### Q3: C1统计按钮组去哪了？
**A**: 统计信息现在显示在状态栏右侧，格式更紧凑

### Q4: 如何快速筛选C1数据？
**A**: 使用工具栏的快捷按钮或右侧的C1筛选下拉框

### Q5: 模型中心/分析中心/C1管理中心在哪？
**A**: 在菜单栏 → 功能中心，目前显示占位对话框，将在后续Phase实现

### Q6: 如何回滚到旧版本？
**A**: 
```bash
Copy-Item "src/v24_app/ui.py.backup_phase1" "src/v24_app/ui.py"
```

### Q7: 快捷键有哪些？
**A**: 
- F5: 刷新赛事
- Alt+F4: 退出
- 更多快捷键将在后续版本添加

### Q8: 状态栏信息太多看不清？
**A**: 状态栏信息已经过优化，如有建议请反馈

## 工作流程

### 典型工作流程 1: 分析赛事

1. 点击"刷新赛事"或按F5
2. 在列表中选择赛事
3. 点击"分析选中"
4. 查看右侧详情面板
5. 根据需要使用C1筛选

### 典型工作流程 2: 查看模型状态

1. 菜单栏 → 工具 → 模型状态总览
2. 查看详情面板中的模型信息
3. 根据需要进行训练或校准

### 典型工作流程 3: 使用功能中心

1. 菜单栏 → 功能中心
2. 选择需要的中心（模型/分析/C1）
3. 在中心窗口中进行操作

## 提示和技巧

### 提示 1: 使用菜单栏快速导航
菜单栏提供了所有功能的入口，按Alt键可激活菜单栏

### 提示 2: 关注状态栏信息
状态栏右侧显示了C1统计、模式和筛选状态，一目了然

### 提示 3: 使用快捷筛选按钮
工具栏的快捷筛选按钮可以快速过滤赛事列表

### 提示 4: 保留用户中心
如果不习惯新界面，可以继续使用传统的用户中心

### 提示 5: 查看关于对话框
帮助 → 关于 可以查看当前版本和重构进度

## 故障排除

### 问题 1: 菜单栏不显示
**解决**: 检查是否正确调用了`_build_menu_bar()`

### 问题 2: 占位对话框一直显示
**解决**: 这是正常的，功能中心将在后续Phase实现

### 问题 3: 状态栏信息不更新
**解决**: 检查`_update_c1_stats()`方法是否正常调用

### 问题 4: 导入错误
**解决**: 运行测试脚本 `python test_ui_phase1.py`
